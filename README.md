# Excel Sales Analysis Project

## 📌 Project Overview

This project focuses on analyzing sales data using Microsoft Excel to extract meaningful business insights and support decision-making.

## 📊 Dataset

* Source: [Vrinda Store data]
* Total Records: 31048
* Features: order id, custid,data,age,month , Sales, Revenue, Region, ship 

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs

## 🔍 Key Steps Performed

1. Data Cleaning (handled missing values, formatting)
2. Data Transformation
3. Created Pivot Tables
4. Built interactive dashboard

## 📈 Key Insights

* Identified top-performing products
* Analyzed monthly revenue trends
* Found highest sales region
* Detected low-performing categories
